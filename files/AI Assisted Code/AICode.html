AI Assisted Coding Assignment

Introduction

In this assignment, you will create your own version of the Connections game inspired by the New York Times.

How to Play

Try to find words that match or belong together.

Select FOUR words that form a group and submit them.

You are only allowed 4 mistakes.

You win by finding all groups before running out of mistakes.

Example

The words Tassel, Gown, Cap, Diploma all belong under the category Graduation Gear.

AI Chatbot Options

  • ChatGPT
  • Gemini
  • Copilot

Tips

  • Be descriptive when prompting the AI.
  • Work in small steps.
  • Expect to debug and revise your code.

Requirements

You must create the following files:

  • index.html
  • style.css
  • script.js

You must also write a one-page reflection explaining:

  • Which AI chatbot(s) you used
  • The quality of the code
  • How much debugging was required
